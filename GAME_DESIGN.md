# 黄金矿工 (Gold Miner) Web版 - 游戏设计文档

## 1. 游戏概述
复刻经典游戏“黄金矿工”，玩家通过控制摆动的钩爪抓取地下的金矿、钻石等物品，在规定时间内达到目标分数即可过关。

## 2. 核心玩法功能
- **钩爪摆动**：钩爪会自动在 0 到 180 度之间左右摆动。
- **发射钩爪**：按下按键后，钩爪沿当前角度伸出。
- **抓取判定**：
  - 钩爪碰撞到物体时，判定抓取成功，开始回拉。
  - 钩爪碰撞到边界（左、右、下）时，判定抓空，空钩回拉（速度较快）。
- **回拉机制**：
  - 不同物品有不同的“重量”，影响回拉速度。
  - 金块重，回拉慢；石头重且价值低；钻石极轻且价值高。
- **计分与关卡**：
  - 每一关有目标分数。
  - 每一关有时间限制（例如 60 秒）。
  - 结算时若分数 >= 目标分数，进入下一关；否则游戏结束。

## 3. 游戏元素
### 物品类型
1. **金块 (Gold)**
   - 大金块：分值高 (500)，重量大 (拉得慢)。
   - 中金块：分值中 (250)，重量中。
   - 小金块：分值低 (100)，重量轻。
2. **石头 (Rock)**
   - 大石头：分值极低 (20)，重量极大。
   - 小石头：分值低 (10)，重量大。
3. **钻石 (Diamond)**
   - 分值极高 (600)，重量极轻 (拉得快)。
   - 通常比较小，难抓。
4. **神秘袋子 (Mystery Bag)**
   - 随机奖励（可能是高分、强力水、或者很少的分数，甚至可能是炸药）。
5. **炸药桶 (TNT)**
   - 触碰即爆炸，摧毁周围物品，不但没分还会损失资源。

### 道具 (商店系统 - 关卡间隙)
- **炸药 (Dynamite)**：当抓到不想要的东西（如大石头）时，按下快捷键炸毁该物品，快速收回钩子。
- **生力水 (Strength Polish)**：下一关拉取物品速度加快。

## 4. 控制方式 (键盘操作)
- **下箭头 (↓) 或 空格键 (Space)**：发射钩爪。
- **上箭头 (↑)**：使用炸药（仅当钩爪抓着东西且拥有炸药时有效）。

## 5. 排行榜与数据存储 (新增)
- **本地存储**：使用 `localStorage` 保存排行榜数据。
- **榜单逻辑**：记录分数前十名的玩家。
- **新纪录交互**：
  - 游戏结束时，如果总分足以进入前十名。
  - 弹出模态框恭喜玩家，并提供输入框输入姓名。
  - 保存后刷新显示排行榜。
- **排行榜展示**：在游戏主菜单或游戏结束画面展示 Top 10。

## 6. 音效系统 (新增)
- 使用 Web Audio API 生成简单音效，无需外部素材，确保即开即用：
  - 发射钩子声。
  - 抓取到物品的碰撞声。
  - 拉取物品时的持续声音（重物低沉，轻物轻快）。
  - 金币/得分结算声。
  - 爆炸声。
  - 游戏结束/过关音乐。

## 7. 技术栈
- **核心**：HTML5 Canvas + JavaScript (原生 ES6+)。
- **样式**：CSS3。
- **数据**：LocalStorage。
- **音频**：Web Audio API (Synthesizer)。
